PWA Service Worker Update Notification

Implementazione notifica aggiornamento PWA con banner visibile

📋 Panoramica Implementazione

🎯 Obiettivo

Quando l'app viene aperta e c'è un aggiornamento disponibile del service worker, mostrare una notifica chiara e visibile all'utente con la possibilità di aggiornare immediatamente.

✅ Funzionalità Implementate

  • Banner di notifica fisso in alto
  • Rilevamento automatico aggiornamenti service worker
  • Bottone "Aggiorna Ora" per applicare l'update immediatamente
  • Controllo periodico aggiornamenti (ogni ora)
  • Animazione slide-down per l'apparizione del banner
  • Design responsive (mobile e desktop)

🔧 Dettagli Tecnici

📄 index.html

1. Banner HTML (dopo body tag)

<div id="update-notification" class="fixed top-0 left-0 right-0 bg-gradient-to-r from-blue-600 to-blue-700 text-white shadow-lg z-50 hidden">
    <div class="container mx-auto px-4 py-3 flex items-center justify-between">
        <!-- Icon + Text -->
        <button id="update-reload-button">Aggiorna Ora</button>
    </div>
</div>

2. CSS Animation (nello style tag)

@keyframes slideDown {
    from { transform: translateY(-100%); opacity: 0; }
    to { transform: translateY(0); opacity: 1; }
}

3. JavaScript Logic

  • showUpdateNotification() - Mostra il banner
  • registration.update() - Controllo periodico ogni ora
  • postMessage('SKIP_WAITING') - Forza l'aggiornamento
  • controllerchange event - Reload automatico

⚙️ service-worker.js

1. Install Event

self.addEventListener('install', event => {
    // ... cache logic ...
    self.skipWaiting(); // ← Force immediate activation
});

2. Message Listener

self.addEventListener('message', (event) => {
    if (event.data && event.data.type === 'SKIP_WAITING') {
        self.skipWaiting();
    }
});

3. Activate Event

self.addEventListener('activate', event => {
    // ... cleanup old caches ...
    return self.clients.claim(); // ← Take control immediately
});

🧪 Istruzioni per il Test

📱 Test su Android/Mobile

  1. Apri l'app PWA già installata
  2. Modifica il codice (es: incrementa versione cache)
  3. Deploy le modifiche
  4. Chiudi completamente l'app (swipe via)
  5. Riapri l'app → Dovresti vedere il banner blu in alto
  6. Clicca "Aggiorna Ora" → L'app si ricarica con la nuova versione

💻 Test su Desktop (Chrome)

  1. Apri Chrome DevTools (F12) → Tab "Application"
  2. Vai su "Service Workers"
  3. Spunta "Update on reload"
  4. Modifica service-worker.js (cambia CACHE_NAME)
  5. Ricarica la pagina (F5)
  6. Deseleziona "Update on reload"
  7. Ricarica di nuovo → Dovresti vedere il banner
  8. Clicca "Aggiorna Ora" per applicare l'update

🔍 Verifica nel DevTools

  • Console: Controlla i log "🔄 Nuova versione disponibile!"
  • Application → Service Workers: Verifica la versione cache attiva
  • Elements: Ispeziona il banner con id="update-notification"
